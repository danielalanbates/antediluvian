# A4 — Skeptical alpha audit ("would millions play this?")

Method: launch v0.2.0 client, play the first 15 minutes, screenshot hubs and
combat, judge against WoW-Classic feel. Verdicts below are updated per pass.

## Pass 1 — 2026-07-22 (post A1–A3 implementation)
- [ ] Visual: 8 terrain families visible and distinct at the inn hub outskirts
- [ ] Visual: mob silhouettes differ at a glance (plans + tusks/sail/tail barbs)
- [ ] Visual: builder hairstyles render on preview AND on remote players
- [ ] Perf: retina fps still ~59 with 1,200 formations/act (was 500)
- [ ] First 15 min: Gate-of-Eden prologue flows without dead ends
- [ ] Combat feel: hit feedback, floating damage, death readable
- [ ] Audio: by-ear mix check (still pending Daniel since C12)

## Honest gaps that block "millions would play this" (post-alpha queue)
1. Combat depth: no talent trees UI parity audit since C16; cooldown feel unpolished.
2. Social: no trade, bank, mail, party/shared credit, dungeons, battlegrounds
   (deliberate post-alpha, but they gate mass retention).
3. Art: procedural variety ≠ hand-authored beauty; hero landmarks (capitals,
   act finale sets) deserve bespoke art passes.
4. Onboarding: no tutorialization beyond quest text; first-session UX untested
   with a fresh player.
5. Server: local-spawned single shard; real multiplayer needs a hosted server
   + the launcher pointed at it.
